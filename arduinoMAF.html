Arduino


¿Qué es Arduino? Es una plataforma de hardware y software libre utilizada para construir dispositivos electrónicos interactivos. Está compuesta por una placa electrónica programable, como la Arduino Uno, y un entorno de programación (IDE) fácil de usar. Fue diseñada para que estudiantes, aficionados y profesionales puedan crear prototipos o proyectos tecnológicos sin necesidad de tener conocimientos avanzados en electrónica.

Características

  • Hardware abierto
  • Fácil de programar
  • Interfaz USB
  • Entradas y salidas digitales/analógicas
  • Compatible con muchos módulos y sensores
  • Bajo costo
  • Amplia comunidad y documentación
  • Modelos variados


Uso de Arduino en la Robótica: Permite controlar y automatizar los componentes de un robot de forma sencilla y económica. Gracias a su flexibilidad, se ha convertido en la base de muchos proyectos educativos, experimentales y hasta profesionales. Puede controlar el movimiento, hacer lectura de sensores, automatización de tareas, comunicarse con otros dispositivos y puede usarse para proyectos de robótica educativa y para hacer prototipos de robots reales.

Interfaz: Llamada IDE (Integrated Development Environment), sus partes son:
  • Verificar/Subir (Verify/Upload): Verifica el código (sin subirlo) o lo sube al Arduino, el check verifica errores, y la flecha sube el código al microcontrolador.
  • Seleccionar placa y puerto (Select Board & Port): Aquí eliges qué placa Arduino estás usando (como Uno, Mega, etc.) y el puerto USB al que está conectada.
  • Abrir monitor serial (Open Serial Monitor): Abre una ventana para ver datos que el Arduino envía, útil para ver valores de sensores o depurar.
  • Abrir trazador serial (Open Serial Plotter): Muestra los datos del Arduino en forma de gráfica en tiempo real, ideal para interpretar valores analógicos.
  • Sketchbook: Acceso rápido a tus proyectos guardados. Cada “Sketch” es como un archivo o programa de Arduino.
  • Administrador de placas (Board Manager): Permite instalar o actualizar el soporte para diferentes placas Arduino o compatibles.
  • Administrador de librerías (Library Manager): Aquí puedes añadir librerías que expanden las funciones del Arduino (como controlar sensores, pantallas, etc.).
  • Depurador (Debugger): En la versión 2.0 permite hacer depuración paso a paso del código (para algunas placas compatibles).
  • Buscar (Search): Sirve para buscar palabras o funciones dentro del código abierto.

Prototipo Robótico de Arduino Ocupado en Clase: El Robot Educativo MAF-01 es un prototipo móvil diseñado con Arduino y equipado con dos ruedas motrices (2WD), un sensor ultrasónico HC-SR04 para detección de obstáculos y una estructura acrílica de doble nivel. Este robot es ideal para introducirse al mundo de la robótica y la programación, ya que permite comprender el funcionamiento de sensores, motores y lógica de control. Gracias a su sencillez y funcionalidad, se convierte en una excelente herramienta para el aprendizaje práctico de la electrónica y la automatización.

Componentes de un Sistema de Arduino:
  • Sensores: Permiten la entrada de información al sistema, captan información del mundo físico y la convierten en señales eléctricas que la placa puede interpretar (ultrasónico, de luz, de temperatura, infrarrojo)
  • Actuadores: Permiten la salida de información o la respuesta del sistema, ejecutan acciones como movimientos físicos al recibir información de la placa de control (motores, servomotores, luces LED, bocinas)
  • Placa de control: Cerebro del sistema, procesa los datos que llegan de los sensores, ejecuta el cargado en su memoria, y decide qué accionas tomar (Arduino Uno, Arduino Mega, Raspberry Pi)
  • Cables: Permiten la comunicación y alimentación entre componentes, el flujo de electricidad y datos entre los sensores, conectores, actuadores y las placas (Cables macho- macho, macho-hembra, hembra hembra, Protoboards para pruebas sin soldar)

Glosario

Actuadores: Componentes que ejecutan acciones físicas (como motores o servos) en respuesta a órdenes del sistema. Se refiere a conjuntos de datos extremadamente grandes y complejos que no pueden gestionarse ni analizarse fácilmente con las herramientas tradicionales de procesamiento de datos, en particular las hojas de cálculo. Incluyen datos estructurados (bases de datos), datos no estructurados (publicaciones sociales) y conjuntos de datos mixtos (lenguaje para IA).

Arduino: Plataforma de hardware y software libre usada para crear dispositivos electrónicos interactivos, muy popular en la robótica educativa.

Automatización: Proceso mediante el cual se utilizan sistemas automáticos para realizar tareas sin intervención humana directa.

Calidad de vida: Aspecto que mejora gracias a la robótica, al facilitar tareas, reducir riesgos y aumentar la eficiencia en distintas áreas.

Comunidad: Grupo de usuarios que comparte conocimientos, recursos y proyectos relacionados con Arduino y robótica.

Control Remoto: También llamado mando a distancia, controla diferentes dispositivos electrónicos y eléctricos para que realicen ciertas acciones.

Educación: Uso de robots y plataformas como Arduino para enseñar programación, fomentar la creatividad y preparar para el futuro laboral.

Hardware libre: Tecnología cuyo diseño es accesible y modificable por cualquier persona para aprender, compartir o mejorar.

Hogar: Espacio donde los robots se aplican en limpieza, asistencia personal y automatización de sistemas domésticos.

IDE (Entorno de Desarrollo Integrado): Programa que permite escribir, cargar y depurar el código que se ejecuta en placas Arduino.

Industria: Sector económico que aplica robots para optimizar procesos de producción, como soldadura, pintura o embalaje.

Inteligencia artificial (IA): Tecnología que permite a las máquinas simular capacidades humanas como el razonamiento, aprendizaje y toma de decisiones.

IRremote: Es una herramienta que permite la recepción de señales infrarrojas, por lo que permite comunicarnos con nuestro control remoto.

Luz Infrarroja: Luz emitida por el control remoto, que es una frecuencia que está por debajo de la sensibilidad de nuestros ojos y es imperceptible.

Medicina: Área donde se usan robots para cirugías asistidas, rehabilitación, diagnóstico y asistencia a personas mayores.

Movilidad autónoma: Capacidad de un robot o vehículo para desplazarse por su entorno sin control humano directo.

Placa programable: Componente físico de Arduino que permite conectar sensores, actuadores y programarla para controlar un robot.

Programación: Proceso de escribir instrucciones que indican a un robot cómo debe comportarse o responder ante estímulos.

Prototipo: Modelo experimental o inicial de un robot o dispositivo, usado para pruebas antes de su versión final.

Robot: Máquina controlada por una computadora que imita acciones humanas o animales para ejecutar tareas específicas.

Robótica: Rama de la ingeniería y ciencia que diseña, construye y programa robots para realizar tareas automáticas o semiautomáticas.

Sensores: Dispositivos que detectan información del entorno (como luz, temperatura o distancia) y la envían al robot para actuar.

Transporte: Campo donde se usan vehículos autónomos, drones de entrega y sistemas inteligentes para mejorar la movilidad.
